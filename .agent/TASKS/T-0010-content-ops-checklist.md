# T-0010: Content Ops Checklist（0代码同学可执行）

## Goal
输出非技术同学可直接执行的内容运营流程文档，覆盖新增/修改/校验/提交流程。

## Scope
- In-scope:
  - 内容更新 SOP
  - 提交前校验命令
  - 常见报错排查
  - 分支与提交命名规范
- Out-of-scope:
  - CMS 实际接入
  - 自动化审批系统

## Acceptance Criteria
- [x] 提供新闻/论文/成员/项目四类更新步骤
- [x] 提供统一校验与预览命令
- [x] 提供常见错误排查
- [x] 提供最小提交流程模板
- [x] RUN 证据留痕
