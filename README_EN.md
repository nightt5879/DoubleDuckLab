# 🦆🦆 DoubleDuckLab (DDLab)
<p align="right">
  <a href="./README.md">中文</a> | English
</p>
A modern academic lab website template built with **Astro**.

Designed for **EE / CS research groups** who want a clean, fast, and easily maintainable website.

---

## Features

* Built with Astro
* Static site generation
* Markdown / JSON content
* Responsive UI
* Dark / Light mode
* Easy deployment

---

## Quick Start

```bash
git clone https://github.com/YOURNAME/doubleducklab.git
cd doubleducklab
npm install
npm run dev
```
